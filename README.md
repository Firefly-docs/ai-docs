# AI Docs

Firefly AI 板块文档仓库，包含以下三个子版块：

- **应用软件 / Applications** — AI 应用软件相关文档
- **工具链 / Dev Toolchain** — AI 开发工具链相关文档
- **模型广场 / Model Zoo** — AI 模型广场相关文档

## 目录结构

```
ai-docs/
├── zh/                    # 中文文档
│   ├── 应用软件/           # AI 应用软件
│   ├── 工具链/             # AI 开发工具链
│   └── 模型广场/           # AI 模型广场
├── en/                    # English Docs
│   ├── Applications/      # AI Applications
│   ├── Dev Toolchain/     # AI Dev Toolchain
│   └── Model Zoo/         # AI Model Zoo
└── README.md
```

## 贡献指南

- 中文文档放在 `zh/` 目录下，英文文档放在 `en/` 目录下
- 文档使用 Markdown 格式
- 图片资源放在对应板块目录下的 `images/` 子目录中
