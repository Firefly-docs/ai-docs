# 使用指南

这里是一段使用说明文本。

## 基本用法

```python
from 测试模块 import 测试类

# 这里是一行注释
测试实例 = 测试类(参数一="测试值")
结果 = 测试实例.测试方法("这里是一个输入文本")
print(结果)
```

## 高级用法

```python
from 测试模块 import 测试类, 高级功能

# 这里是一行注释
测试实例 = 测试类(
    参数一="测试值",
    参数二=42,
    参数三=True,
)

# 这里是一行注释
for 输出 in 测试实例.流式方法("这里是一个输入文本"):
    print(输出, end="")
```

## 命令行使用

```bash
测试命令 --参数一 值一 --参数二 值二 --输入 "这里是一个测试输入"
```
